# Commands:

`whisper "public/sample.mp4" --model tiny.en --output-dir ./public --max_words_per_line 10 --word_timestamps True`

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```

**Upgrade Remotion**

```console
npm run upgrade
```
